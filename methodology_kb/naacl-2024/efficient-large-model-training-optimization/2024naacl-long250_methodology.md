# Impossible Distillation for Paraphrasing and Summarization: How to Make High-quality Lemonade out of Small, Low-quality Model

**Source**: https://aclanthology.org/2024.naacl-long.250/

## [POSITIVE] Paraphrastic Proximity
The observation that a sentence and its paraphrases tend to lie on a proximal subspace in the pretrained LM distribution, allowing multiple paraphrases to be generated by constraining the LM with informative context.

**Delta**: Enables generation of high-quality paraphrases from small LMs (GPT2-XL) that cannot perform the task directly.
**Condition**: Applicable when using off-the-shelf LMs (e.g., GPT2) with sufficiently informative context (1-5 sentences) and appropriate decoding temperature (0.7).

**Evidence**: "We hypothesize and verify the paraphrastic proximity intrinsic to pre-trained LMs (e.g., GPT2), where paraphrases occupy a proximal subspace in the LM distribution."

## [POSITIVE] Pair Generation with Contextual Constraints
Generating a large pool of candidate (source, paraphrase) pairs by sampling multiple next sentences conditioned on a contextual constraint (1-5 sentences), then enumerating all combinations.

**Delta**: Produces a diverse data pool; DIMPLE dataset (4.2M pairs) outperforms 13x larger datasets in diversity.
**Condition**: Used in the initial data generation stage; requires sampling k=100 generations per context with nucleus sampling and temperature 0.7.

**Evidence**: "Our dataset, as a purely synthetic corpus generated from 1.5B LMs, shows better metrics in all measures than state-of-the-art datasets: ParaBank that is 13 times larger than ours."

## [POSITIVE] Semantic Equivalence Filter (NLI-based)
Using an off-the-shelf NLI model (RoBERTa-large-WANLI) to filter pairs where bidirectional entailment score exceeds a threshold (τ_semantic = 0.75), ensuring semantic preservation.

**Delta**: Critical for removing noisy pairs; contributes to high faithfulness scores in human evaluation (2.55 vs ChatGPT's 2.37).
**Condition**: Applied after pair generation; threshold τ_semantic = 0.75.

**Evidence**: "We define a binary filter using a small NLI model as a critic, and discard the pairs that do not achieve the entailment score over the threshold τ_semantic."

## [POSITIVE] Dissimilarity Filter (ROUGE-L + Tree Edit Distance)
Filtering pairs based on token overlap (ROUGE-L < τ_rouge = 0.75) and syntactic difference (Tree Edit Distance > τ_TED = 12) to ensure surface-form dissimilarity.

**Delta**: Promotes lexical and syntactic diversity; DIMPLE achieves TED-F of 29.84 vs ChatGPT-Para's 20.15.
**Condition**: Applied after semantic filter; thresholds τ_rouge = 0.75, τ_TED = 12.

**Evidence**: "For token overlap, we filter the pairs with higher ROUGE-L than a threshold τ_rouge. To gauge the syntactic difference, we filter based on their tree edit distance (TED)."

## [POSITIVE] Diversity Filter (Graph-based Deduplication)
Removing duplicate pairs by grouping entailing pairs (using NLI) and keeping only the one with the highest entailment score per connected component.

**Delta**: Improves corpus diversity; DIMPLE shows higher lexical diversity (H3=22.46 vs ChatGPT-Para's 21.41).
**Condition**: Applied after dissimilarity filter; uses NLI model to detect entailment between pairs from the same context.

**Evidence**: "To remove the duplicate pairs and promote diversity, we employ an additional critic f_diversity."

## [POSITIVE] Self-Distillation
Fine-tuning the student model on its own high-quality paraphrases (generated by the student itself and filtered by critics) to amplify performance.

**Delta**: Improves BERT-iBLEU by 3.1 points on MRPC (from 64.0 to 67.1).
**Condition**: Applied after initial student training; requires generating paraphrases from the student model and re-filtering with critics.

**Evidence**: "BERT-iBLEU in this case degrades by 3.1 from Impossible-T5, indicating the importance of self-distillation in amplifying model capability."

## [POSITIVE] Using GPT2-scale Teacher (1.5B) instead of LLM
Using a small, off-the-shelf LM (GPT2-XL, 1.5B parameters) as the teacher instead of a large LLM like GPT-3, relying on paraphrastic proximity and critic filtering.

**Delta**: Outperforms models distilled from ChatGPT (175B) on multiple benchmarks; e.g., BERT-iBLEU 67.1 vs 62.9 on MRPC.
**Condition**: Applicable when a large LLM is unavailable or too costly; requires critic models for filtering.

**Evidence**: "Our model with 770M parameters consistently outperforms strong baselines, including models distilled from ChatGPT, and sometimes, even ChatGPT itself."

## [NEUTRAL] Critic Filtering on ChatGPT-generated Data (Ablation)
Applying the same set of critics (semantic, dissimilarity, diversity) to filter ChatGPT-generated paraphrases, then training a student model.

**Delta**: Improves over unfiltered ChatGPT-Para (BERT-iBLEU 64.1 vs 62.9) but still worse than Impossible-T5 (67.1).
**Condition**: When using ChatGPT as teacher; less sample-efficient due to misalignment with critic criteria.

**Evidence**: "BERT-iBLEU on MRPC marks 64.1, improving over the original ChatGPT-Para but still falling behind Impossible-T5."

## [POSITIVE] Low Decoding Temperature (0.7) for Paraphrase Generation
Using a decoding temperature of 0.7 during pair generation to balance diversity and sample efficiency, based on analysis of paraphrastic proximity.

**Delta**: Increases ratio of semantically equivalent pairs while maintaining diversity (Self-BLEU ~32).
**Condition**: Used during pair generation stage; temperature too low leads to near-identical sentences.

**Evidence**: "The ratio of semantically equivalent pairs dramatically increases as the temperature decreases. ... we set the decoding temperature τ_temp = 0.7 to balance the diversity and sample efficiency."

## [POSITIVE] Generalization to Sentence Summarization via Filter Redefinition
Adapting the Impossible Distillation framework to sentence summarization by redefining the critics (e.g., adding a length filter to ensure output is shorter than input).

**Delta**: Outperforms baselines on Gigaword; BERTScore-F1 85.1 vs ChatGPT's 84.8; human faithfulness 2.53 vs 2.44.
**Condition**: When the target task changes from paraphrasing to summarization; requires task-specific filter definitions.

**Evidence**: "IMPOSSIBLE DISTILLATION successfully generalizes to summarization, only by plugging in the redefined composition of filters."

## [POSITIVE] Syntactic Control via Constituency Tree Conditioning
Training a controllable paraphrase model by conditioning on the constituency tree of the target paraphrase, enabling syntax-controlled generation.

**Delta**: Outperforms ChatGPT in syntax-controlled generation; iBLEU 11.2 vs ChatGPT's 9.1.
**Condition**: When syntactic control is desired; requires parsing target paraphrases to extract constituency trees.

**Evidence**: "Impossible-T5 outperforms baselines across all metrics except ROUGE-L. Notably, the syntax conformity of ChatGPT is substantially poor."
