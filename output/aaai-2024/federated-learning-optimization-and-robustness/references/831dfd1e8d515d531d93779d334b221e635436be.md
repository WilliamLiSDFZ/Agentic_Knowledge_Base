# Adapted Weighted Aggregation in Federated Learning

**Source**: [https://www.semanticscholar.org/paper/831dfd1e8d515d531d93779d334b221e635436be](https://www.semanticscholar.org/paper/831dfd1e8d515d531d93779d334b221e635436be)

**TLDR**: Introduces a federated learning algorithm with an adapted weighted aggregation mechanism sensitive to client dataset quality for improved performance.

## Abstract

This study introduces FedAW, a novel federated learning algorithm that uses a weighted aggregation mechanism sensitive to the quality of client datasets, leading to better model
performance and faster convergence on diverse datasets, validated using Colored MNIST.
