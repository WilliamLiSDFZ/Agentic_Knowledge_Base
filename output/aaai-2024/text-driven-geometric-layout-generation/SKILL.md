# Text Driven Geometric Layout Generation

This skill involves generating and refining structured 2D layouts (like bounding boxes or wireframes) from text descriptions, using methods such as non-autoregressive transformers and training-free control mechanisms. Its primary tasks include urban layout regeneration and consistent multi-element scene generation, with applications in graphic design, architectural planning, and UI/UX prototyping.
