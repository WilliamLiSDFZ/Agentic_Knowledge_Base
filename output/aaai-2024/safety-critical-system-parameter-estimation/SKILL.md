This skill involves estimating parameters for systems where failure is unacceptable, using methods like runtime monitoring and neural network-based control to detect and correct for unmodeled dynamics or safety constraint violations. Its core tasks are ensuring system safety through real-time verification and adaptive model repair, directly applied to autonomous vehicles, robotics, and industrial control systems.
