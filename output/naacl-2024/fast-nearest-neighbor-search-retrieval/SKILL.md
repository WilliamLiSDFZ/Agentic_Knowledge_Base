# Fast Nearest Neighbor Search Retrieval

This skill covers retrieval-augmented generation (RAG) methods that enhance large language models by retrieving and incorporating relevant external documents to improve the accuracy and grounding of their responses. The primary tasks include dense retrieval and information fusion, with key applications in knowledge-intensive question answering and reducing model hallucinations.
