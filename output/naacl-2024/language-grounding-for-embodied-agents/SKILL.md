# Language Grounding For Embodied Agents

This skill involves generating and interpreting natural language expressions that uniquely identify objects or locations in a shared physical environment, a core task for enabling robots to follow instructions like "hand me the blue mug on the left." It applies methods from computational linguistics and computer vision to establish human-robot common ground, which is critical for applications in collaborative assembly, domestic service, and navigation assistance.
