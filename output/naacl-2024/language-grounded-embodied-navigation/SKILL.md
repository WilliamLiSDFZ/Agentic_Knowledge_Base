# Language Grounded Embodied Navigation

This skill covers methods for grounding natural language instructions in visual environments to enable embodied agents to navigate and interact with objects, primarily using multimodal models (e.g., VLMs, LLMs) for tasks like object navigation, instruction synthesis, and UI automation. Applications include training on-device agents, improving robustness against attacks, and achieving zero-shot generalization in open-world settings.
