# Information Extraction Knowledge Graph Methods

This skill covers methods for extracting structured knowledge from text, including entity linking without pre-defined candidates, knowledge graph completion using contextual generation, and unified frameworks that combine tasks like relation extraction and event detection. It also involves adapting these techniques for multilingual and code-mixed data through instruction tuning and parameter-efficient fine-tuning, with applications in detecting salient entities and populating knowledge graphs.
