# Testing the Effect of Code Documentation on Large Language Model Code Understanding

**Source**: [https://aclanthology.org/2024.findings-naacl.66/](https://aclanthology.org/2024.findings-naacl.66/)

**TLDR**: Investigates how code documentation and other properties affect LLMs' ability to understand and generate code or documentation.

## Abstract

AbstractLarge Language Models (LLMs) have demonstrated impressive abilities in recent years with regards to code generation and understanding. However, little work has investigated how documentation and other code properties affect an LLM’s ability to understand and generate code or documentation. We present an empirical analysis of how underlying properties of code or documentation can affect an LLM’s capabilities. We show that providing an LLM with “incorrect” documentation can greatly hinder code understanding, while incomplete or missing documentation does not seem to significantly affect an LLM’s ability to understand code.
