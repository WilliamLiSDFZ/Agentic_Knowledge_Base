# Financial Risk Narrative Analysis Datasets

This skill involves applying natural language processing (NLP) and machine learning methods to extract, classify, and quantify risk-related signals from unstructured news text. The core tasks include named entity recognition for companies, sentiment analysis, and topic modeling to build predictive models for financial, operational, or reputational risk assessment.
