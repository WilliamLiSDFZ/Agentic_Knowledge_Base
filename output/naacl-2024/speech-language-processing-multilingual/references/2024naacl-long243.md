# Universal NER: A Gold-Standard Multilingual Named Entity Recognition Benchmark

**Source**: [https://aclanthology.org/2024.naacl-long.243/](https://aclanthology.org/2024.naacl-long.243/)

**TLDR**: Introduces Universal NER, a community-driven project to create gold-standard multilingual NER benchmarks.

## Abstract

AbstractWe introduce Universal NER (UNER), an open, community-driven project to develop gold-standard NER benchmarks in many languages. The overarching goal of UNER is to provide high-quality, cross-lingually consistent annotations to facilitate and standardize multilingual NER research. UNER v1 contains 19 datasets annotated with named entities in a cross-lingual consistent schema across 13 diverse languages. In this paper, we detail the dataset creation and composition of UNER; we also provide initial modeling baselines on both in-language and cross-lingual learning settings. We will release the data, code, and fitted models to the public.
