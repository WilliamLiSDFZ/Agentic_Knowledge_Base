# 全参数微调存在硬天花板

Run8 Branch2 用全参数微调 + LinearWarmup 策略，8个变体（不同正则化、不同dropout、集成/非集成）全部在 0.26~0.34 区间，无一突破 0.26。

## 证据

| 变体 | Log Loss | 差异点 |
|------|----------|--------|
| top3 | 0.2631 | 基础方案 |
| top4 | 0.2863 | 变异 |
| top6 | 0.2886 | +集成 |
| top7 | 0.2924 | +CosineAnnealing+集成 |
| top5 | 0.2868 | +HybridStylo头 |
| top11 | 0.3132 | +HybridStylo+LS |
| top12 | 0.3368 | 无调度器 |
| top9 | 0.3016 | 3模型集成 |

**无论怎么调参、加集成、换头，全参数微调的天花板就是 ~0.26。** 只有改变微调策略（部分解冻）才能突破。

## Actionable Guidance

- 如果全参数微调连续3个变体无法突破某个分数，立即切换到部分解冻策略
- 不要在已饱和的策略上浪费演化步数
